Girl CodeRunners 2019

A summer camp on basics of coding for middle school girls in grade 6 through 8.

In this camp, the girls will learn about
  1. Coding to create animations and stories using scratch.mit.edu
  2. Coding to create computer games using scratch.mit.edu
  3. Coding robots using LEGO Mindstroms EV3
  4. Website design using Kompozer
This camp is supported by Texas Workforce Commision through a grant to the University of Texas at San Antonio
